# Session 5 Cue Card Reader

A dependency-free, mobile-friendly reader for the Session 5 speaker cue cards.

## Controls

- Left/right arrows or swipe to change cards
- Space to start or pause the timer
- R to reset the current timer
- F for fullscreen
- Optional auto-advance when a card's allotted time expires

## GitHub Pages

Publish the contents of this folder from the root of a repository, or place it
under a repository's `docs/` folder and select that folder in Settings > Pages.
